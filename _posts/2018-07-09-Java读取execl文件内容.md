使用Idea编辑器操作execl文件，首先要引入apache的一个jar包，叫做poi。

# 涉及到知识点

| 知识点                                                       | 疑问                                         |
| ------------------------------------------------------------ | -------------------------------------------- |
| idea如何引入jar包， 打开 File -> Project Structure，单击 Modules -> Dependencies -> "+" -> "Jars or directories"， 选择硬盘上的jar包  ，Apply -> OK | 这样只是在这一个项目中导入，之后可以直接使用 |
